# Venkatesh Dongala - Personal Portfolio Website

Welcome to the GitHub repository for my personal portfolio website. This project showcases my skills, projects, and experience as a data scientist and AI enthusiast. The website is designed to be a clean, modern, and interactive representation of my professional profile.

### Features

- **Interactive UI:** Smooth animations, hover effects, and a dynamic particle background create an engaging user experience.
- **Fully Responsive Design:** The layout is optimized for all screen sizes, from mobile phones to desktop computers.
- **Dynamic Content:** Features a typing animation in the hero section to introduce my roles.
- **Project Showcase:** Highlights my key data science and machine learning projects with descriptions and links to their GitHub repositories.
- **Skills & Experience:** Clearly outlines my technical skills and professional experience in a clean, easy-to-read format.

---

### Tech Stack

This portfolio is built with modern web technologies, focusing on performance and a clean codebase.

- **HTML5:** For the core structure and content.
- **Tailwind CSS:** For all styling, enabling a responsive and utility-first design.
- **JavaScript:** For interactive elements like the typing animation, particle background, and smooth scrolling.
- **Lucide Icons:** For clean and modern icons throughout the site.
